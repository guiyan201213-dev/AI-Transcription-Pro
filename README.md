# AI 听写 Pro (AI Dictation Pro) 🎧

**AI 听写 Pro** 是一款极简、优雅且功能强大的 H5 智能听写辅助工具。它旨在解决学生在居家学习时，无法独立完成词语、生字或外语单词听写的痛点。

---

## ✨ 项目亮点

- **零成本上手**：无需下载安装，打开浏览器即可在手机或电脑上使用。
- **智能语音交互**：基于原生 Web Speech API，响应速度快，无需联网合成音频。
- **沉浸式体验**：界面清爽无广告，UI 设计符合现代审美，操作逻辑直观。
- **完全自主可控**：自由调节语速、书写间隔与重复次数，模拟真实的考场或课堂听写环境。

## 🛠️ 主要功能

- [x] **词语列表输入**：支持空格、逗号或换行符快速分隔词语。
- [x] **语速调节 ($0.5x$ - $2.0x$)**：针对不同熟练程度，提供灵活的朗读速度。
- [x] **书写间隔自定义**：为长词或生僻字留出充足的等待时间。
- [x] **重复朗读机制**：可设置每个词语重复 1-3 遍，确保听得清、记得牢。
- [x] **响应式布局**：完美适配移动端（iOS/Android）及桌面端浏览器。

## 🚀 快速开始

1. **输入词语**：在输入框内粘贴你需要听写的词语或单词。
2. **个性化设置**：拖动进度条调整你喜欢的语速和间隔时长。
3. **开始听写**：点击底部的 **“开始听写”** 按钮。
4. **控制进度**：听写过程中可以随时停止。

## 💻 技术实现

本项目采用原生前端技术栈开发，追求极致的加载速度：

- **HTML5**: 构建页面结构与交互控件。
- **CSS3**: 使用 Flexbox 与 Grid 布局，结合 Glassmorphism（渐变/毛玻璃）视觉风格。
- **JavaScript**: 处理业务逻辑、状态管理及语音流控制。
- **Web Speech API**: 调用浏览器内置合成引擎进行文本转语音（TTS）。
